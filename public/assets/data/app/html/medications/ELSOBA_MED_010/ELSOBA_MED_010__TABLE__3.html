Major Bleeding Classifications Table
Severity Definition
   
Type 1
  • Any requirement to reduce heparin infusion
  • Any PRBC transfusion due to bleeding
Type 2
  • Requirement to reduce heparin infusion and PRBC transfusion
  • Requirement of nonsurgical procedure (including endoscopic interventions) to stop bleeding
Type 3
  • Any life-threatening bleeding requiring PRBC transfusion with or without surgical intervention for control of bleeding
  • Any intracranial hemorrhage (except microbleeds/ hemorrhagic transformation)
  • Any cardiac tamponade
Type 4
  • Any fatal bleeding
