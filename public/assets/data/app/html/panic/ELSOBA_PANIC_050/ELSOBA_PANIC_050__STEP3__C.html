The pumphead is disengaged or uncoupled, the pump is not properly seated.

A.
Decrease RPMs to 0 and clamp the return line to prevent backflow.
B.
Reposition the pumphead.
C.
Restart the pump. Remove the last clamp on the return line only when the minimum RPM is reached to prevent backflow [depending from IFU insert link to almanac section].
D.
Strictly monitor the circuit and the patient.
E.
Document the event and related procedures in the patient's chart.
F.
If applicable, report the issue in the ELSO registry.
This concludes No Flow.
